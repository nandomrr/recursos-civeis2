# Recursos Civeis - Guia Interativo

Este e um site estatico. O arquivo principal e `index.html`.

## GitHub Pages

1. Entre em https://github.com/new
2. Crie um repositorio publico chamado, por exemplo, `recursos-civeis`
3. Envie todos os arquivos desta pasta para o repositorio
4. Abra `Settings` > `Pages`
5. Em `Build and deployment`, escolha `Deploy from a branch`
6. Em `Branch`, escolha `main` e pasta `/root`
7. Salve e aguarde o link ficar disponivel

O link normalmente fica assim:
`https://SEU_USUARIO.github.io/recursos-civeis/`

## Netlify

1. Entre em https://app.netlify.com/drop
2. Arraste esta pasta inteira para a area de upload
3. Aguarde a publicacao
4. Copie o link terminado em `netlify.app`

## Vercel

Opcao mais simples:

1. Suba esta pasta para um repositorio no GitHub
2. Entre em https://vercel.com/new
3. Importe o repositorio
4. Mantenha as configuracoes padrao
5. Clique em Deploy

O Vercel reconhece este projeto como site estatico, porque existe um `index.html` na raiz.
